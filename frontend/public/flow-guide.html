üöÄ Quick Start Guide

1. Start the Servers

# Frontend (Already running ‚úì)

# HTML Pages Server
./start-collaborative-server.sh

2. Generate QR Code

Create a QR code pointing to: http://YOUR_LOCAL_IP:8080/word-submit.html

3. Choose Your QR Display

Option 1: Minimalist

Clean, centered design with word count. Best for professional settings.

Option 2: Dynamic Grid

Split screen with live word preview and stats. Most engaging!

Option 3: Immersive

Fullscreen with floating QR. Perfect for large venues.

Admin Dashboard

Control panel for managing words and generating the song.

4. Test the Flow

  1. Open QR display on one screen
  2. Open word-submit.html on your phone
  3. Submit a test word
  4. Watch it appear on the display
  5. Open admin dashboard
  6. Click "Generate Song"
